## 概要
本リポジトリは、JavaScript（Vanilla JS）を用いて作成したToDoアプリです。
ユーザーがタスクを追加・完了・編集・削除できる基本的な機能を実装しており、
DOM 操作やイベントハンドリングを自前で実装しています。

## 使用技術
- JavaScript（Vanilla JS）
- Bootstrap
- HTML / CSS

## 機能一覧
- タスクの追加
- タスクの完了状態の切り替え（チェック）
- タスクの編集
- タスクの削除
- 全タスク数・完了済みタスク数・未完了タスク数のリアルタイム表示
- ローカルストレージを利用したタスクの永続化
